# **App Name**: Yatharth Abhifly Wholesale

## Core Features:

- Home Page Showcase: Prominently display the company's value proposition with clear 'Call Now' and 'WhatsApp' CTAs.
- Product Category Display: Categorize and display products for easy browsing without pricing information.
- "Why Choose Us" Section: Highlight key benefits like wholesale pricing and reliable supply chain to build trust.
- Contact & Inquiry Forms: Implement a simple inquiry form and direct contact options, including click-to-call and WhatsApp integration.
- Mobile Responsiveness: Ensure the website is fully responsive on all devices for optimal user experience.
- SEO Optimization: Implement SEO-friendly structure and practices to improve website visibility.
- Dealer Inquiry AI: If a prospective dealer submits the "Become a Dealer" inquiry form, an AI tool will look for similar queries and associated resolution paths from existing dealer relationship management documentation to guide an efficient reply.

## Style Guidelines:

- Primary color: Dark blue (#224369) to evoke trust and reliability, aligning with the company's professional image.
- Background color: Clean white (#F5F5F5) for a professional and uncluttered feel.
- Accent color: Muted steel grey (#778899) for subtle highlights and a modern industrial look.
- Body and headline font: 'PT Sans', a humanist sans-serif for readability and a modern touch.
- Clean and organized layout with clear sections and intuitive navigation.
- Use professional, simple icons to represent product categories and key benefits.
- Subtle transitions and hover effects to enhance user engagement without being distracting.
